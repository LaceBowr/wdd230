White Space & Clean Design:

Whitespace- is the area between the elements on a web page. Elements can mean typography, and icons. It is used to balance elements and create natural flow for the viewer.

Rule of Thirds:

The Rule of Thirds- means dividing an image or layout into thirds both vertically and horizontally, creating a grid with nine equal-sized sections.

Repetition:

Repetition- means repeating a single element many times in a design. In this example you see each image repeats the same kind of landscape.
